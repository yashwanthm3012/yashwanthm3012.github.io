Understanding Federated Learning

A privacy-preserving machine learning technique

Federated Learning (FL) is a decentralized approach to training machine learning models. Instead of collecting all the data in a central server, the model is trained locally on devices, and only the model updates (not raw data) are sent to a central server.

This technique is especially useful in scenarios where data privacy is crucial — such as in healthcare, finance, or mobile devices.

Key Advantages:

  • Data remains on the user's device
  • Improved privacy and security
  • Reduces bandwidth usage

Real-World Example:

Google uses Federated Learning in Gboard (its keyboard app) to learn how people type and suggest better predictions — all while keeping user data private.
